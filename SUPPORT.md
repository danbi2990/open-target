# Support

If you run into a bug, unexpected path resolution, or a target that should have
opened but did not:

- Open an issue on GitHub:
  - `https://github.com/danbi2990/open-target/issues`
- Include the line content you tried to open, if possible
- Mention whether the target was a Markdown link, URL, file path, or
  `path:line[:column]` reference
- Mention whether the file was inside a multi-root workspace

This makes it much easier to reproduce navigation and path-resolution issues.
